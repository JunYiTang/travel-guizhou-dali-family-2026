荔波 vs 大理玩水 / 親水比較

核心差異：荔波是溪谷、瀑布、湖泊、漂流的「山水型玩水」，記憶點強但受天氣與安全管制影響較大；大理是洱海散步、短程騎行、SUP/皮划艇的「慢親水」，舒服放空但刺激感較低。
項目荔波 / 貴州大理 / 洱海
玩水型態小七孔、臥龍潭、水上森林、鴛鴦湖划船、水春河漂流。龍龕/才村洱海散步、生態廊道短程騎行、SUP/皮划艇、雙廊看海。
刺激感較高，尤其漂流。較低，偏舒服、放空、活動感。
親子舒適度中等。景觀強，但景區步行、濕滑、排隊與天氣變數較多。較高。當地移動短，活動可彈性取消或改散步咖啡。
10歲小孩適合度小七孔、鴛鴦湖很適合；漂流需看身高、年齡、水況與業者規定。散步、騎行很適合；SUP/皮划艇需教練、救生衣與風浪穩定。
天氣風險7月雨季，水色漂亮但步道濕滑；漂流可能因水量或天氣停開。7月紫外線強、午後雨、洱海風浪影響水上活動。
推薦給誰想讓孩子有「哇」的自然景觀和玩水記憶。想住得舒服、慢慢看水、古城美食與親水兼顧。
我的推薦若目標是「玩水有感」，荔波勝。若目標是「舒服親子旅行 + 一點玩水」，大理勝。

貴州 V2 Day 4–7觀山湖 / 貴陽北 / 市政府一帶觀山湖市政府亞朵、觀山湖開元名庭、全季/希爾頓歡朋/麗呈/美豪類型不一定要住甲秀樓/噴水池老城區，親子休息可能較吵。
大理 V2 Day 1–4龍龕碼頭 / 才村，或洱海生態廊道周邊龍龕碼頭親子房、才村碼頭家庭房、大理恬舍洱海酒店、山海溪月酒店不要硬追一線海景；暑假可能超預算，二線安靜院落更穩。
大理 V2 Day 5–7大理古城東門/南門外圍古城東門/南門外圍家庭房、雙床、含早餐、洗衣房、可寄放行李避開古城酒吧街、主街正中、無電梯又要搬行李的民宿。

住宿結論：若以「住得舒服、行程順、預算內穩定」為最高目標，選大理 V2。若選貴州，請選貴州 V2，並把荔波住宿列為最需要早訂和仔細看評價的部分。

飯店評價與親子房篩選規則

  • 評分：攜程/Trip.com 優先 4.7 以上，最好 4.8 以上。
  • 評論數：飯店至少 500 則較穩；民宿至少 200 則較穩。
  • 近半年差評必看，不只看總分。
  • 親子避雷關鍵字：隔音差、潮濕、有味道、熱水不穩、沒電梯、早餐差、圖文不符、離景區其實很遠、親子房太小、加床不舒服。
  • 加分條件：家庭房/親子房、雙床或大床+單床、有洗衣房、有電梯、可寄放行李、早餐穩、叫車方便、不在酒吧街/主街最吵位置。
